# wedo
Script Python de création d'utilisateur en masse pour Wedo à partir d'un fichier CSV de la forme : 

firstname;lastname;email:invite

ou

* firstname = prénom
* lastname = nom
* email = email
* invite = envoyer une invitation (true) ou non (false)

Pensez à changer les valeurs suivantes : 


* csv_file_path = 'users.csv'
* api_token = 'API_TOKEN'
